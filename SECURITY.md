# 安全说明

## API Key

项目开发者不提供公共 API Key。

玩家在页面输入的 API Key 只保存在本机浏览器 localStorage，不写入仓库。请求会经过本地 Node 服务代理，主要用于解决浏览器跨域问题。

不要把真实 API Key 写进：

- `README.md`
- Mod 文件
- Issue
- Pull Request
- 代码注释
- 截图

## Mod 脱敏

提交职业 Mod 前必须删除：

- 真实公司名
- 真实客户名
- 真实供应商名
- 真实同事姓名
- 报价、成本、合同
- 图纸、工艺参数、内部流程编号
- 未公开产品信息
- 内部报告、邮件、聊天记录

## 高风险职业

医疗、法律、金融、危险作业等职业 Mod 必须明确模拟边界，不得把游戏内容写成真实操作指导。

## 报告问题

如果发现泄露密钥、商业机密或其他安全问题，请不要公开扩散，优先联系仓库维护者处理。
