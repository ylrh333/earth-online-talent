# 地球 Online 职业 Mod 模拟器项目规则

## 项目定位
- 开源本地 Web 项目：玩家下载仓库后，配置自己的 AI API Key，加载职业 Mod 进行开放式职业模拟。
- 项目核心不是测评问卷，而是“职业 Mod 框架 + 可玩职业副本 + AI 裁判/NPC”。
- 开发者不提供公共 API Key，不承担玩家 AI 调用成本。
- 第一版先做本地运行，不做登录、不做付费、不做线上上传平台。

## 目录约定
- `docs/`：产品方案、Mod 规范、创作者指南、迭代计划。
- `app/`：本地 Web 游戏框架，包含页面、样式和前端逻辑。
- `career-mods/`：职业 Mod。每个具体岗位一个独立目录。
- `tools/`：本地启动、Mod 校验、构建辅助脚本。

## MVP 约束
- 不做全职业平台，先做 1 个高真实度官方示例 Mod：汽车质量工程师。
- 不引入外部 npm 依赖，保证 `node tools/serve.js` 可直接本地运行。
- API Key 由玩家自己在本地输入；前端只保存到浏览器 localStorage，不写入仓库。
- 本地服务只作为静态文件服务和 AI 请求代理，不保存玩家聊天内容。
- Mod 以结构化文件为主，方便 AI 生成初稿后人工修改。

## 合规
- 不宣称职业诊断，不替代职业咨询、心理测评、投资建议或法律建议。
- 输出报告使用“表现倾向”“可能适合探索”“本局表现”等表达。
- 不采集身份证、手机号、真实姓名、位置、通讯录等敏感信息。
- Mod 不得包含真实公司、客户、供应商、同事、报价、图纸、内部文件或商业机密。
- 涉及危险作业、医疗、法律、金融等高风险职业时，必须明确这是模拟体验，不能当作真实操作指导。
